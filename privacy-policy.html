Privacy Policy

ROXSTAR Eco System respects user privacy and is committed to protecting personal data.

Apps Covered

  • ROX ID
  • Trace It
  • Nyra Browser
  • Roxyy Search

Information We Collect

  • Device information (model, OS version)
  • Firebase-generated identifiers
  • Location data (only when required and with user consent)

Location Usage

Trace It app uses background location only for device tracking and security features. A clear disclosure screen is shown, and users can stop tracking anytime using the Stop Protection button.

ROX ID uses foreground location only while the app is open. Nyra Browser and Roxyy Search do not access location.

Data Sharing

We do not sell or share user data with third parties. All apps communicate securely within the ROXSTAR Eco System using a shared Firebase project.

User Control

  • Users can disable location anytime
  • Users can stop background tracking
  • Search history saving is optional and controlled by user
