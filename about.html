I run a Kannada tech channel called Tech with Roxstar, and I recently launched my first Android app called NetPredictor — now available on the Play Store.

📱 About NetPredictor

NetPredictor is a network intelligence app built to help users check which SIM network works best in their area. It supports Jio, Airtel, Vi, and BSNL with GPS-based accuracy and smart AI suggestions like:

  • Compare SIM1 vs SIM2 signal strength
  • Switch to 4G if 5G is unstable
  • Battery usage per SIM
  • Speed comparison and network suggestions

The app is crowdsourced — meaning, the more users start using it, the smarter and more useful it becomes. My dream is to spread this app across India, especially in rural and low-signal areas where it's needed most.
